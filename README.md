# TerminalAutoPilot

An AI-powered command-line assistant that automates system operations and developer workflows using natural language and voice commands.

## Features
- Voice-enabled commands
- System automation
- Developer workflow automation
- AI-powered responses
- Command execution from natural language

## Technologies Used
- Python
- OpenAI API
- Speech Recognition
- Text-to-Speech

## Installation
pip install -r requirements.txt

## Run
python main.py
